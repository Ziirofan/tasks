# tasks

Une todo list en ligne de commande

## Commandes

```shell
task add "example"
```

Ajoute une nouvelle tache



```shell
task read "1"
```

lit la tache selectionnée



```shell
task update "1" "new text"
```

Met à jour la tache selectionnée



```shell
task done "1"
```

Termine la tache sélectionnée
